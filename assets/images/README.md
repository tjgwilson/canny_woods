# Images

Put website images in this folder.

Suggested structure:

- `assets/images/banner/` for hero and page banner images
- `assets/images/gallery/` for gallery photos
- `assets/images/icons/` for small graphics or logos

Recommended file naming:

- use lowercase
- use hyphens instead of spaces
- keep names descriptive, for example `woodland-canopy-spring.jpg`

Current usage:

- the homepage hero will look for `assets/images/banner/home-hero.jpg`

If that file exists, it can be wired into the design immediately.
