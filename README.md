# Canny Woods

Starter site for the GitHub Pages website hosted from this repository.

## Local structure

- `index.html` is the homepage.
- `styles.css` contains the site styles.
- `script.js` is available for lightweight client-side behavior.

## Publish with GitHub Pages

1. Push this repository to GitHub.
2. In the GitHub repository, open `Settings` -> `Pages`.
3. Set the source to deploy from the main branch root.
4. Save and wait for GitHub Pages to publish the site.
